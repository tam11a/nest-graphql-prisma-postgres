# NestJS, GraphQL, Prisma, Postgres

Boilerplate

## Installation

```bash
npm install
```

## Create Environment File

Copy `template.env` and create `.env` file

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
